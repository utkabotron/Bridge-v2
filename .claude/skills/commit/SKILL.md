# Commit Skill

Сделать `git add -A`, коммит и `git push`.

## Steps

1. Показать что изменилось:
   ```bash
   git status --short
   git diff --stat HEAD
   ```

2. Если изменений нет — сообщить и завершить.

3. Определить сообщение коммита:
   - Если `$ARGS` передан — использовать его
   - Иначе — сформулировать по изменённым файлам (кратко, в стиле `feat/fix/chore`)

4. Коммит и пуш:
   ```bash
   git add -A
   git commit -m "<message>"
   git push
   ```

5. Вывести hash коммита и статус пуша.
